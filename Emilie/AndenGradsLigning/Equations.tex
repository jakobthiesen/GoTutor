\section*{Andengradsligninger og deres Koefficienter}

% Inline math example
Find koefficienter (a, b, c) for 2. grads ligninger. \\
 -Hint: Start med at opstille standard-form, så som:
 \[2x^2 -3x = 2 \Rightarrow 2x^2-3x-2 = 0\] \\
 Altså, forsøg at få 0 på den ene side. Såles er koefficienterne:\\
 \[a = 2, b=-3, c=-2\]
\\
% Display math example
\begin{enumerate}
     \item \[2x^2 + x - 5 = 0\]\\
     \item \[-10x^2=2x+3\] \\
     \item \[x^2 = 5x\] \\
     \item \[x(x+3)=2\]\\
     \item \[(x+3)(x+1) = 1\] \\
     \item \[(x)(x+2) = 0\]
\end{enumerate}
